# learning-lab
Repositorio de estudio y práctica (cursos/módulos). No son proyectos finales.
## Estructura
- `001-programacion-1-2/`: Material de Programación 1 y 2 (ejercicios, notas, tareas).
## Convenciones
- Cada carpeta tiene su propio `README.md` con apuntes y links.
- Nombres: `00X-tema/` para mantener orden cronológico.
- Si un trabajo se vuelve “proyecto para portfolio”, se mueve a un repositorio separado.
## Notas
- No se incluyen datos sensibles ni credenciales.
- Los datasets grandes se reemplazan por links a la fuente (o muestras pequeñas).
